There are times when you need to zip your project, either for backup or whatever. You can always use tar or zip commands. But if you use git as a version control, there is a very simple way to achieve this.
Say hello to git-archive!

It's simple. Type:

git archive -o name.zip HEAD

when in your project directory.

This will automatically zip your project and save the output (-o) to a file called 'name.zip'. Zip is not good enough? No problem, rename the output file to 'name.tar'.

Cheers.
